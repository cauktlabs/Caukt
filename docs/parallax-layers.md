# Parallax layers

Three parallax layers: distant ceiling girders, mid-hall stalls, near-floor crates. Scroll speeds are 0.3x, 0.7x, and 1.0x of the foreground.

## Update 2

Mid-hall layer recycles four stall templates with different signage and steam patterns. Four is enough; a fifth made the repetition more obvious, not less.

## Update 3

Near-floor crates randomize per run using the daily auction seed. Same seed, same crate pattern.
